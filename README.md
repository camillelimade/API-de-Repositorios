# API-de-Repositorios
Esse sistema contribui para a visualização aprimorada dos dados de repositórios de um usuário do GitHub, utilizando da própria API do serviço. Diante disso, ele também contribui com os fins empíricos da minhas expiências na  área de Backend, favorecendo a ampliação da minha atuação profissional.

## Como funciona?
O site funciona de forma que quando o usuário digite o `username`, um input o recebe e um botão por meio de um evento de onclick chama a função getApi() que consiste em receber os dados atríbuidos, buscar o json da API e por meio das variáveis dele exibir isso ao usuário priorizando o fundamental.

## Tecnologias
- Visual Studio Code
- API do GitHub
- JavaScript
